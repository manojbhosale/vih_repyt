About the server:

HIVsubtyper assigns subtype to query genome sequence(s) of HIV-1 virus.
The server uses a novel method based on Markov Chain Model.

Sever also hosts tools for Markov order determination of the nucleotide
sequences and clustering of the sequences at given order.

HIV-1 Subtyping server has been developed at Bioinformatics Centre, University of Pune.
